User Stories
============
```
As a user
I want to have my thermostat default to 20 degrees
So I can sit in a warm lounge

As a user
I want to increase the temperature of my thermostat
so that I can have a warmer room

As a user
I want to decrease the temperature of my thermostat
So I can chill in a cooler room

As a user
I want to be prevented from decreasing the temperature below 10
So that I don't get hypothermia
```
